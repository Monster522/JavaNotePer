## 习题解析

### 1.集合ArrayList删除功能

```java
public class Test {
    public static void main(String[] args) {
        List<Integer> NumberList = new ArrayList<Integer>();
        NumberList.add(2);
        NumberList.add(4);
        NumberList.add(1);
        NumberList.add(3);
        NumberList.add(5);
        
        for(int i =0;i<NumberList.size();++i) {
            int v = NumberList.get(i);
            if(v%2==0) {
              NumberList.remove(v);
            }
        }
        System.out.println(NumberList);
    }
}

* 运行结果：出现集合越界的情况
```
